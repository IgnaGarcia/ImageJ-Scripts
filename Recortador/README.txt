Manual para el Usuario de Recortador.js


1- Abrir ImageJ

OPCIONAL:
	- Abrir Imagen a Procesar
	- Con la herramienta MultiPoint marcar los centros a recortar

2- Plugins > New > JavaScript

3- Abrir Recortador.js ( Copiar y pegar contenido, si no File > Open )

4- Ejecutar( Ctrl + R )

5- Si ya abrio la imagen seleccionela, sino debera buscarla dandole a Browse(No recomendado para Raw)
	- OK cuando termine de seleccionar

6- Si no marco los centros de los recortes se le advertira que los marque
	- OK cuando termine de marcar

7- Buscar el directorio donde se guardaran los recortes, especificar el nombre de las salidas y el formato
	- Tenga en cuenta de que el nombre se le concatenara al principio un numero
	- OK para continuar

8- Fin, puede cerrar los recortes


OPCIONAL: 
	puede guardar el script desde la ventana del editor de texto de imageJ
	si lo guarda dentro de ImageJ\plugins lo encontrara en el menu de plugins de ImageJ
	Esto evita tener que abrir el editor, y abrir el script
