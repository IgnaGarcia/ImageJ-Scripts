Manual para el Usuario de CustomPlugin.js

1- Copiar y pegar la carpeta Custom en ImageJ\plugins

2- Abrir ImageJ

3- Plugins > Custom > CustomPlugin

4- Se ejecutara el Recortador.js{
	Seleccionar la imagen
	Marcar los puntos recortar y analizar posteriormente
	Ingresar los datos del output

	Repetir
}

5- Se ejecutara el Summarizer.js{
	Seleccionar la imagen
}

Fin
