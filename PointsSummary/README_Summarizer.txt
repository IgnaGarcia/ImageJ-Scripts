Manual para el Usuario de Points_Summary.js


1- Abrir ImageJ

OPCIONAL:
	- Abrir la primer imagen Imagen a Procesar

2- Plugins > New > JavaScript

3- Abrir Summarizer.js ( Copiar y pegar contenido, si no File > Open )

4- Ejecutar( Ctrl + R )

5- Si ya abrio la imagen seleccionela, sino debera buscarla dandole a Browse(No recomendado para Raw)
	- OK cuando termine de seleccionar

6- Fin, puede guardar las tablas como csv en File > Save As...
    - El formato de salida es .csv

OPCIONAL: 
	puede guardar el script desde la ventana del editor de texto de imageJ
	si lo guarda dentro de ImageJ\plugins lo encontrara en el menu de plugins de ImageJ
	Esto evita tener que abrir el editor, y abrir el script
